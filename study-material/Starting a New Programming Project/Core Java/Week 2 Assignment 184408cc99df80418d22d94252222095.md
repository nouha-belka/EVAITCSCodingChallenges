# Week 1 Assignment: Java Fundamentals

In this assignment, you will demonstrate your understanding of Java's basic concepts covered in Week 1. Complete all tasks below:

## Part 1: JVM Understanding

- Write a short essay (250 words) explaining how the Java Virtual Machine enables platform independence
- Create a diagram showing the compilation and execution process of a Java program

## Part 2: Basic Syntax Practice

- Write a simple Java program that prints "Hello, World!" to the console
- Create a program that demonstrates proper use of semicolons, brackets, and code indentation
- Add appropriate comments to your code explaining each section

## Part 3: Variables and Data Types

- Create a program that declares and initializes one variable of each primitive data type
- Demonstrate type conversion between different numeric data types
- Write code that shows the difference between primitive and reference data types

## Submission Guidelines

- Submit all code files with .java extension
- Include comments explaining your code
- Submit your essay as a separate PDF document

## Grading Criteria

- Code functionality and correctness (40%)
- Proper syntax and formatting (30%)
- Documentation and comments (20%)
- JVM concept understanding (10%)
