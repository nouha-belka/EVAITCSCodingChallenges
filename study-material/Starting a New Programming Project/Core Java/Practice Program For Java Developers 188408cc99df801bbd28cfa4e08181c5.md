# Java Coding Practice Problems

## Basic Problems (1-10)

1. Reverse a String
    
    ```java
    // Write a program to reverse a string without using built-in reverse function
    // Input: "hello"
    // Output: "olleh"
    ```
    
2. Find Factorial
    
    ```java
    // Calculate factorial of a number using both recursive and iterative approaches
    // Input: 5
    // Output: 120
    ```
    
3. Check Prime Number
    
    ```java
    // Determine if a given number is prime
    // Input: 17
    // Output: true
    ```
    
4. Fibonacci Series
    
    ```java
    // Generate Fibonacci series up to n terms
    // Input: 6
    // Output: 0 1 1 2 3 5
    ```
    
5. Palindrome Check
    
    ```java
    // Check if a string is palindrome
    // Input: "racecar"
    // Output: true
    ```
    
6. Count Vowels and Consonants
    
    ```java
    // Count number of vowels and consonants in a string
    // Input: "Hello World"
    // Output: Vowels: 3, Consonants: 7
    ```
    
7. Armstrong Number
    
    ```java
    // Check if number is Armstrong number
    // Input: 153
    // Output: true (1^3 + 5^3 + 3^3 = 153)
    ```
    
8. Find GCD
    
    ```java
    // Find Greatest Common Divisor of two numbers
    // Input: 48, 36
    // Output: 12
    ```
    
9. Binary to Decimal Conversion
    
    ```java
    // Convert binary number to decimal
    // Input: "1010"
    // Output: 10
    ```
    
10. Sum of Digits
    
    ```java
    // Calculate sum of digits in a number
    // Input: 12345
    // Output: 15
    ```
