# Week 4 Project

# Library Management System Project

Create a comprehensive Library Management System that incorporates Week 3's advanced Java concepts.

## Project Requirements

### 1. Exception Handling

- Implement custom exceptions for book checkout failures
- Handle invalid input scenarios gracefully
- Create a robust error logging system

### 2. Collections Framework

- Use ArrayList to maintain book inventory
- Implement HashMap for user management
- Utilize TreeSet for maintaining sorted book categories

### 3. Generics

- Create generic methods for handling different types of library items
- Implement generic collections for flexible data management

### 4. SQL Integration

- Design and implement a MySQL database for persistent storage
- Create tables for books, users, and transactions
- Implement basic CRUD operations

### 5. Design Patterns

- Implement Singleton pattern for database connection
- Use Factory pattern for creating different types of library items
- Apply Observer pattern for notification system

### 6. SOLID Principles

- Follow Single Responsibility Principle in class design
- Implement Open/Closed Principle for extensibility
- Apply Interface Segregation for different user types

## Deliverables

- Source code with proper documentation
- MySQL database schema
- Unit tests for core functionality
- README file with setup instructions

## Evaluation Criteria

- Proper implementation of all Week 3 concepts
- Code quality and organization
- Error handling robustness
- Adherence to SOLID principles
